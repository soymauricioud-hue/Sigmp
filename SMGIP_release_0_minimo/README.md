# SMGIP — Repositorio inicial de implementación documental

Este repositorio inicia como espacio de fragmentación, trazabilidad y control documental del SMGIP.

No inicia como carpeta de redacción libre.

## Regla principal

Antes de redactar documentos sustantivos, debe existir una base mínima de trazabilidad:

1. `01_release_0/00_release_0_minimo.md`
2. `01_release_0/01_manifiesto_inicial.md`
3. `01_release_0/02_matriz_fragmentacion_0.15_preliminar.md`
4. `01_release_0/03_mapa_trazabilidad_15.19_preliminar.md`
5. `01_release_0/04_mapeo_inicial_26.21_preliminar.md`

Con esos archivos, se desbloquea la redacción preliminar de:

- 0.1 Documento maestro / Propuesta técnica unificada
- 0.2 Exposición de motivos
- 0.3 Teoría de cambio
- 0.4 Alcance, principios y fases
- 0.5 Glosario maestro y taxonomía operativa
- 0.6 Matriz de dependencias documentales

## Cómo usar este repo

1. Copia tus documentos base en `00_fuentes/`.
2. Revisa el Release 0 mínimo.
3. Usa los prompts de `03_prompts_codex/` para pedirle a Codex que redacte un documento a la vez.
4. Guarda cada salida en `02_documentos/00_nucleo_constitucional/`.
5. Audita cada documento antes de pasar al siguiente.
