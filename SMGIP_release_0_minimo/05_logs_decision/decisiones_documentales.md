# Decisiones documentales

## Decisión 001 — Inicio por Release 0 mínimo

Se decide iniciar el repositorio como espacio de fragmentación, trazabilidad y control documental, no como carpeta de redacción libre.

## Decisión 002 — Desbloqueo limitado

El Release 0 mínimo desbloquea únicamente la redacción preliminar del núcleo 0.1–0.6.

No desbloquea operación, reglamento, sanciones, inspección plena, licencias, CNVR formal ni formatos.
