# RELEASE 0-MIN — Desbloqueo del núcleo 0.1–0.6

## Objetivo

Crear la base mínima de trazabilidad para iniciar la redacción lineal del núcleo constitucional, semántico y doctrinal del SMGIP.

## Naturaleza

Este release no implementa el SMGIP en campo.

Este release habilita la redacción controlada de los primeros documentos rectores, evitando que la IA redacte sin contexto, invente estructura o active capacidades institucionales antes de tiempo.

## Documentos incluidos

- `01_manifiesto_inicial.md`
- `02_matriz_fragmentacion_0.15_preliminar.md`
- `03_mapa_trazabilidad_15.19_preliminar.md`
- `04_mapeo_inicial_26.21_preliminar.md`

## Documentos que este release permite empezar

- 0.1 Documento maestro / Propuesta técnica unificada
- 0.2 Exposición de motivos
- 0.3 Teoría de cambio
- 0.4 Alcance, principios y fases
- 0.5 Glosario maestro y taxonomía operativa
- 0.6 Matriz de dependencias documentales

## Capacidades que NO habilita

Este release no habilita:

- reglamento municipal;
- inspección plena;
- sanciones;
- licencias;
- QR obligatorio;
- CNVR formal;
- control pleno de criadores;
- acreditación pet friendly plena;
- cofinanciación regulada;
- operación territorial;
- Fase I operativa.

## Veredicto

Con este release, el núcleo 0.1–0.6 puede redactarse en estado E2 preliminar y avanzar a E3 después de auditoría contra las fuentes base.
